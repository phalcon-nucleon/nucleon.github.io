\Neutrino\Http\Controller is an abstract class who extends \Phalcon\Mvc\Controller.
This class allow you to use Middleware.

If you don't need to use middleware, you can extends directly \Phalcon\Mvc\Controller

In \Phalcon\Mvc\Controller __construct is final.
It's a performance hack with Zephir. If you need to build a controller with the __construct method, you just must implement \Phalcon\Mvc\ControllerInterface, and build you own controller.
